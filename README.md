# 🚀 Interactive Counter Master

A sophisticated, highly customizable "Vanilla" JavaScript web application. This project was developed as part of the **start2impact Full Stack Software Developer Master**, designed to demonstrate advanced DOM manipulation and logic flow.

## 🔗 Live Demo
Check out the live application here: **(https://matteomallia.github.io/progetto-counter/)**

## 🎯 The Brief
The project required the creation of a functional counter application built exclusively with pure JavaScript.
* **Dynamic Interface**: 100% of the UI elements are generated via JavaScript.
* **Core Functionality**: Users can increment and decrement values starting from 0.
* **Code Quality**: Strict separation of concerns (HTML/CSS/JS) and modular function design.

## ✨ Key Features
* **Triple Mode System**: Choose between **Classic Mode** (minimalist), **Fun Mode** (with interactive rewards), and **Maintenance Mode** (future updates).
* **Real-time Customization**: Change the background and counter box colors using integrated color pickers.
* **Persistent Settings**: User style preferences are saved locally using the **Web Storage API**, ensuring they remain even after a page refresh.
* **Gamified Milestones**: Visual rewards (GIFs) appear at specific numeric thresholds (1, 5, 10, 25, 50, 100).
* **Interactive Easter Eggs**:
    * **Level 23**: The entire page shakes temporarily using custom CSS keyframes.
    * **Level 55**: The control buttons swap positions dynamically.

## 🛠️ Technical Stack
* **HTML5**: Minimal semantic skeleton with **Open Graph** meta tags for professional social sharing.
* **CSS3**: Modern layout using **Flexbox**, CSS Variables, and responsive Media Queries for mobile optimization.
* **JavaScript (ES6+)**: Pure "Vanilla" logic, asynchronous timing with `setTimeout`, and event-driven architecture.

## 📂 Folder Structure
The project follows a clean, professional architecture:
* `/js`: Application logic and DOM manipulation (`script.js`).
* `/css`: Responsive styles and animations (`style.css`).
* `/img`: Graphical assets and achievement GIFs.
* `index.html`: Main entry point.
