# 🚀 Super Counter Master - JS Advanced Project

Un'applicazione JavaScript "Vanilla" altamente personalizzabile sviluppata per il Master di **start2impact**.

## 🎯 Funzionalità
- [cite_start]**Creazione Dinamica del DOM**: L'intera interfaccia è generata via script[cite: 1498].
- **3 Modalità di Utilizzo**: Classic, Fun (con Easter Eggs) e Dev Mode.
- [cite_start]**Personalizzazione Real-time**: Cambio colori di sfondo e box con persistenza in `localStorage`[cite: 1636].
- [cite_start]**Easter Eggs**: Effetti speciali (shake, inversione bottoni) a soglie specifiche[cite: 1561].
- **Responsive Design**: Ottimizzato per Desktop e Mobile.

## 🛠️ Tecnologie Utilizzate
- [cite_start]**JavaScript (ES6+)**: Logica, manipolazione DOM e Web API[cite: 1534].
- [cite_start]**CSS3**: Animazioni personalizzate e Flexbox per il layout[cite: 1880].
- [cite_start]**HTML5**: Struttura semantica minima di base[cite: 1448].

## 📂 Struttura del Progetto
- `/js`: Logica applicativa (script.js).
- `/css`: Stili e animazioni (style.css).
- `/img`: Asset grafici e GIF animate.
- `index.html`: Punto di ingresso dell'applicazione.
